Foundation Overview

Foundation is a highly customizable front end framework built with sass (an extension of css), html, and JS. Sass stands for "Syntactically Awesome Style Sheets" and adds features such as nesting and variables to stylesheets and allows for easy customization of foundation componenets. Files created using SASS are processed to css files, and is why this page utilizes foundation without error. Below are just some examples of built in foundation features.
