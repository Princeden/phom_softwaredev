Foundation Overview

Foundation is a highly customizable front end framework built with sass (an extension of css), html, and JS. Sass stands for "Syntactically Awesome Style Sheets" and adds features such as nesting and variables to stylesheets and allows for easy customization of foundation componenets. Files created using SASS are processed to css files, and is why this page utilizes foundation with cdn without error. Below are just some examples of built in foundation features.

Grids!

The most interesting feature of Foundations is the XY Grid classes that help make organization easier. Defaults is that the entire width of the page is 12 cells. Shown below are somes examples from documentation (with added colors for clarity). Theres a lot you can do combining grids with other features .

What is the purpose of medium-6 large-4 and all these numbers??

The XY grid is built to adapt to different screen sizes, and for any page the width of the entire page will be 12 cells. So having a medium-6 and a large-4 allows you to control how large each cell will be on a different device. The 12/6/4 cell will be 6 cells wide on a medium page and 4 cells wide on a large page. You can see the medium-6 size change at play if you shrink the window from full screen.

On a small screen it will default full width 12 cells and the cells will stack on top.

With grids you can have both padding and margins
